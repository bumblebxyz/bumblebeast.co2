Features

The BumbleBeast is an all-in-one desktop machine for professional level manufacturing. It combines a large build area with incredible strength and precision, at an attainable price. Its modular design allows for 3D printing and laser cutting attachments. It is controlled wirelessly with a Microsoft Surface Pro 4 tablet complete with Autodesk Fusion 360 Ultimate CAD/CAM and Mach4 motion control software.

    About Us

    BumbleBeast was born of Hao Cao and Mike Muzio's frustration at using the tools in the Harvey Mudd College (HMC) machine shop - we just wanted one machine that could make functional parts with minimal hassle. We were then accepted into HMC INQ, the HMC startup incubator, in July 2017.
